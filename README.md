## Overview

Falling-Object-Dodge is a fast-paced arcade game built with Python and Pygame. You control a colored player block and must dodge falling blocks unless they match your color. Every 5 points, the falling speed increases.

---

## Features

**Gameplay**
<ul>
<li>Color-matching collision system</li>
<li>Increasing difficulty every 5 score</li>
<li>Randomized falling objects</li>
</ul>

**Architecture**
<ul>
<li>Simple, readable structure</li>
<li>Single-file design</li>
</ul>

**Performance**
<ul>
<li>Optimized rendering loop</li>
<li>Stable 60 FPS gameplay</li>
</ul>

**Dependencies**
<ul>
<li>Python standard library</li>
<li>External: <code>pygame</code></li>
</ul>
